# Курсовая работа: Предсказание биологической активности химических соединений

Модели машинного обучения для предсказания эффективности химических соединений против вируса гриппа по молекулярным дескрипторам.

## Целевые переменные
- **IC50** — концентрация ингибирования вирусной активности на 50%
- **CC50** — цитотоксическая концентрация
- **SI = CC50 / IC50** — индекс селективности

## Структура репозитория
| Файл | Описание |
|------|----------|
| eda_Гимадеева.ipynb | Разведочный анализ данных |
| Регрессия_IC50.ipynb | Регрессия для IC50 |
| Регрессия_CC50.ipynb | Регрессия для CC50 |
| Регрессия_SI.ipynb | Регрессия для SI |
| Классификация_IC50.ipynb | Классификация IC50 > медианы |
| Классификация_CC50.ipynb | Классификация CC50 > медианы |
| Классификация_SI_median.ipynb | Классификация SI > медианы |
| Классификация_SI_8.ipynb | Классификация SI > 8 |
| Аналитический отчет Гимадеева.docx | Аналитический отчёт |

## Использованные модели
Ridge, Lasso, Random Forest, Gradient Boosting, Logistic Regression, Decision Tree
